1. INTRODUCTION                                   *ai-gitcommit-intro*

ai-gitcommit.nvim generates commit messages from staged git diff.

Supported providers:
  - OpenAI
  - Anthropic
  - GitHub Copilot (OAuth)

Requirements:
  - Neovim 0.11+
  - curl

==============================================================================
2. CONFIGURATION                                  *ai-gitcommit-config*

IMPORTANT:
Old flat config is not supported anymore. You must set `provider` and
`providers`.

Minimal setup: >
  require("ai-gitcommit").setup({
    provider = "openai",
    providers = {
      openai = {
        api_key = vim.env.OPENAI_API_KEY,
      },
    },
  })

Full setup: >
  require("ai-gitcommit").setup({
    provider = "openai", -- required

    providers = {
      openai = {
        api_key = vim.env.OPENAI_API_KEY,
        model = "gpt-4o-mini",
        endpoint = "https://api.openai.com/v1/chat/completions",
        max_tokens = 500,
      },

      anthropic = {
        api_key = vim.env.ANTHROPIC_API_KEY,
        model = "claude-haiku-4-5",
        endpoint = "https://api.anthropic.com/v1/messages",
        max_tokens = 500,
      },

      copilot = {
        model = "gpt-4o",
        endpoint = "https://api.githubcopilot.com/chat/completions",
        max_tokens = 500,
        client_id = nil, -- optional override
      },
    },

    languages = { "English", "Chinese", "Japanese", "Korean" },
    prompt_template = nil,

    context = {
      max_diff_lines = 500,
      max_diff_chars = 15000,
    },

    filter = {
      exclude_patterns = {
        "%.lock$",
        "package%-lock%.json$",
        "yarn%.lock$",
        "pnpm%-lock%.yaml$",
        "%.min%.[jc]ss?$",
        "%.map$",
      },
      exclude_paths = {},
      include_only = nil,
    },

    keymap = nil,
    auto = {
      enabled = true,
      debounce_ms = 450,
    },
  })

Custom prompt placeholders:
  - {language}
  - {extra_context}
  - {staged_files}
  - {diff}

Provider config validation:
  - providers.<name>.model must be non-empty string
  - providers.<name>.endpoint must be non-empty string
  - providers.<name>.max_tokens must be > 0

==============================================================================
3. COMMANDS                                       *ai-gitcommit-commands*

:AICommit [{context}]                              *:AICommit*
  Generate commit message in gitcommit buffer.

:AICommit login <provider>                        *:AICommit-login*
  Login for OAuth providers.

:AICommit logout <provider>                       *:AICommit-logout*
  Logout provider.

:AICommit status                                  *:AICommit-status*
  Show all provider status.

:AICommit status <provider>                       *:AICommit-status-provider*
  Show one provider status.

`<provider>` must be one of:
  - openai
  - anthropic
  - copilot

Notes:
  - OpenAI uses API key config. `:AICommit login openai` returns guidance.
  - Anthropic and Copilot support OAuth login.

==============================================================================
4. PROVIDERS                                     *ai-gitcommit-providers*

OpenAI:
  Configure `providers.openai.api_key`.

Anthropic:
  Configure API key, or run: >
    :AICommit login anthropic

Copilot:
  Run: >
    :AICommit login copilot

  OAuth data is stored in: >
    stdpath("data")/ai-gitcommit/copilot.json

==============================================================================
5. TROUBLESHOOTING                               *ai-gitcommit-troubleshooting*

"No provider configured"
  Set `provider = "openai"|"anthropic"|"copilot"` in setup.

"OpenAI API key not configured"
  Set `providers.openai.api_key`.

"Not authenticated"
  Run login for your provider:
    :AICommit login anthropic
    :AICommit login copilot

"Not in a gitcommit buffer"
  The command works only in commit message buffer (opened by `git commit`).
